## Changelog

### [1.0.0] - 2026-01-05

#### Initial Public Release (Memo2496 Only)

- MuseFuse framework implementation for Memo2496 dataset.
- Integrated ProtoAlign, ReliaPseudo, and TriDistill modules with default paper hyperparameters.
- Added training script `train_musefuse.py` and core library under `lib/`.
- Provided basic documentation, license, and dependency specification for Figshare release.
